# VoidVibes - Discord Bot
In Russian:
Дискорд бот разработанный по заказу для одноименного сервера. Содержит в себе модераторские функции, функции экономики(за искл. казина) и брака с разводом. 
Позволяет работать с верификацией и выдачей роли по нажатию кнопки. Пока предназначен только для одного сервера, в дальнейшем будет оптимизирован под работу на нескольких серверах
Работа над ботом началась: ~09.02.26 года
Выход v0.1: 15.02.2026 года

In English:
A Discord bot developed on order for the eponymous server. It includes moderator functions, economy features (excluding the casino), and marriage with divorce.
It allows working with verification and role assignment via button click. Currently, it is only designed for a single server, but will be optimized for multi-server support in the future.
Work on the bot began around February 26, 2026.
Release of version 0.1: February 15, 2026.
